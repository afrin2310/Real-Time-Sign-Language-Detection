# 🤟 Real-Time Sign Language Detection System

An AI-powered real-time Sign Language Recognition application built using Python, Streamlit, OpenCV, and MediaPipe. This project helps detect and interpret hand gestures instantly through a webcam, aiming to improve communication accessibility for hearing and speech-impaired individuals.

---

## 📌 Project Overview

Communication barriers can make daily interactions difficult for people who rely on sign language. This project uses Artificial Intelligence and Computer Vision to recognize hand gestures in real time and convert them into understandable outputs.

The application captures live video through a webcam, detects hand landmarks using MediaPipe, and processes gesture recognition efficiently through an interactive Streamlit web interface.
